## ✨ Features Overview

### Resizable Plot
- Adjust width (400-2000 px) and height (300-1500 px)
- Data points auto-scale when plot resizes
- Responsive container design
- CSS + JavaScript-based resizing

### Enhanced Legend
- Show/hide toggle via checkbox
- Click entries to toggle/isolate series
- Hover to highlight related data
- Professional styling with color indicators

### Interactive Toolbar
- 📷 Download as PNG
- 🔍 Zoom controls
- 🔄 Pan tool
- ↺ Reset view
- ⚙️ Settings

### Hover Tooltips
- Shows X, Y, Color, and Selection values
- Multi-line formatted text
- Appears on data point hover
- Disappears when moved away

### Other Features
- Color-code by any field (vcc, temp, status, etc.)
- Split plots by selection
- Keyboard shortcuts (scroll, drag, etc.)
- Download plot as PNG
- Responsive design
