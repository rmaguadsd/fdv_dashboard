# Plot Features Summary

## ✨ New Interactive Plot Features (Just Added!)

### 1. **Resizable Plot** ↔️ 📏
- Adjust width (400-2000 px) and height (300-1500 px)
- Click "Apply Size" button to resize
- Data points automatically adjust and reflow
- Direct CSS resize also supported

### 2. **Legend Controls** 📋
- **Show Legend** checkbox → Toggle legend visibility
- **Legend Clickable** checkbox → Click legend entries to:
  - Toggle visibility of a data series
  - Double-click to isolate series
  - Right-click for other options
  - Hover to highlight related points

### 3. **Interactive Toolbar** 🛠️
- **📷 Download**: Save plot as PNG
- **🔍 Zoom**: Box select and zoom to region
- **🔄 Pan**: Click-drag to move around
- **↺ Reset**: Return to full view
- **Scroll wheel**: Quick zoom in/out

### 4. **Hover Tooltips** 💬
- Position cursor over data point
- Shows exact X, Y, and color dimension values
- No more estimating from grid lines

### 5. **Responsive Plot**
- Automatically reflows when resized
- Data points scale with container
- Maintains aspect ratio

### 6. **Color-Code by Any Field** 🎨
- Dropdown: vcc, temp, status, plane, pagetype, pr, wl, blk, tm, fdv, value
- Each unique value gets distinct color
- Legend shows color → value mapping

### 7. **Split by Selection** 📊
- Create separate subplots for each selected item
- Cleaner, less cluttered visualization
- Easier to spot individual patterns

---

## Quick Start

1. **Generate Plot**
   - Select X and Y fields
   - Optional: Choose color-by field
   - Optional: Check "Split plots by selection"
   - Click "Generate Plot"

2. **Resize Plot** (if needed)
   - Enter Width: e.g., 1200
   - Enter Height: e.g., 800
   - Click "Apply Size"

3. **Use Legend**
   - Uncheck "Show Legend" to hide (more space)
   - Check "Legend Clickable" to filter by clicking
   - Hover legend entry to highlight data

4. **Explore Data**
   - Hover over points to see values
   - Click zoom 🔍 to zoom to region
   - Scroll wheel to zoom in/out
   - Click reset ↺ to return to full view

5. **Export**
   - Click 📷 button to download PNG

## User Tips

| Task | How To |
|------|--------|
| See better details | Increase Width/Height in size controls |
| Focus on one data series | Click it in legend (or double-click to isolate) |
| Hide legend to maximize plot | Uncheck "Show Legend" |
| Know what each color means | Hover over color entry in legend |
| Save plot for later | Click 📷 button in toolbar |
| Undo zoom | Click ↺ reset button |
| Compare two selections | Split by selection OR color by FDV |

---

## How It Works

1. **Generation**: User clicks "Generate Plot"
2. **Backend**: Plotly creates interactive HTML visualization
3. **Frontend**: HTML embedded in iframe for isolation
4. **Interactivity**: 
   - Hover → Shows custom tooltip
   - Click legend → Toggles visibility
   - Resize controls → Updates iframe height/width
   - Toolbar → Native Plotly tools (zoom, pan, download)
