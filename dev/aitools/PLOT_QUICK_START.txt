┌─ CONTROL LEGEND ──────────────────────────────────────────────────────────┐
│                                                                             │
│  After plot appears, you'll see legend controls:                          │
│                                                                             │
│  ┌─ Legend Controls ─────────────────────────────────────────────────┐    │
│  │ ☑ Show Legend    ☑ Legend Clickable                              │    │
│  └───────────────────────────────────────────────────────────────────┘    │
│                                                                             │
│  Features:                                                                 │
│                                                                             │
│  ☑ Show Legend (checked)      → Legend is visible                         │
│  ☐ Show Legend (unchecked)    → Legend hidden (more plot space)           │
│                                                                             │
│  ☑ Legend Clickable (checked) → Click legend items to filter              │
│    • Click:  Hide/show that data series                                    │
│    • Double-click: Isolate that series (hide all others)                  │
│    • Right-click: Additional options                                       │
│                                                                             │
│  ☐ Legend Clickable (unchecked) → Read-only legend                        │
│                                                                             │

┌─ USE INTERACTIVE FEATURES ────────────────────────────────────────────────┐
│                                                                             │

│  ├─ Tooltip appears showing:                                              │

│  ├─ Option 1: Click-drag on plot background                              │
