Help & Documentation

🚀 Getting Started

  1. Upload your FDV log file using the Upload & Parse tab
  2. The app will parse the log file and generate a CSV with extracted data
  3. View and filter the data in the View Data tab
  4. Create visualizations using the Scatter Plot and CDF Plot tabs
  5. Download the generated CSV or plots for further analysis

📝 Supported Log Formats

The parser supports FDV log files with the following formats:

  • FDV OUTPUT - Functional test data with pass/fail results
  • FDV POLL - Parametric/characterization data with measurements

📊 Available Plots

  • Scatter Plot - X-Y scatter with optional category coloring
  • CDF Plot - Cumulative distribution with split capability

🔑 Key Features

  • Automatic parsing of complex FDV log file formats
  • Extraction of test parameters (VCC, TEMP, WL, BLK, PAGE, etc.)
  • CSV generation with comprehensive data
  • Interactive data browsing and filtering
  • Publication-quality plot generation
  • Statistical summaries of parsed data

📚 Extracted Fields

The parser extracts the following fields from FDV logs:

  • DUT (Device Under Test)
  • Test Name & Test Conditions
  • Pagetype (SLC, MLC, TLC, QLC, etc.)
  • WL (Wordline) and BLK (Block) addresses
  • RBER/Value (Raw Bit Error Rate)
  • VCC, VCCQ, TEMP (Power and Temperature)
  • Status codes and plane operations
  • Deck information (LD, MD, UP)
  • And more...
